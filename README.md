# Jio Hotstar & JioTV Kodi Addons

This repository contains two Kodi addons for streaming content from Jio Hotstar and JioTV.

## Addons Included

1.  **Jio Hotstar (`plugin.video.jiohotstar`):** Browse and watch content from the merged JioHotstar platform.
2.  **JioTV (`plugin.video.jiotv`):** Stream live TV channels using your Jio account.

## Installation

1.  Download the folders `plugin.video.jiohotstar` and `plugin.video.jiotv`.
2.  Zip each folder individually.
3.  In Kodi, go to **Settings > System > Add-ons** and enable **Unknown sources**.
4.  Go to **Add-ons > Install from zip file** and select the zipped files.

## Usage

### JioTV
1.  Open the JioTV addon.
2.  Select **Login to JioTV**.
3.  Enter your Jio mobile number when prompted.
4.  Enter the OTP received via SMS.
5.  Once logged in, the channel list will appear.

### Jio Hotstar
1.  Open the Jio Hotstar addon.
2.  Browse through the available categories or search for content.
3.  Click on a title to play.

## 2026 Update Note
These addons have been updated to support the new **JioStar** unified domain and the latest **JioTV** endpoints (`media.jio.com`). 

## Requirements
- Kodi 19+ (Matrix, Nexus, Omega)
- Inputstream Adaptive (version 19.0.0+)
- An active Jio account/number for JioTV.
